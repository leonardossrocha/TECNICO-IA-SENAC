## Apresentação da UC

Esta Unidade Curricular tem como objetivo introduzir a linguagem de programação Python de forma essencialmente prática, com foco em resolução de problemas e construção progressiva do código. 

Aprenderemos a transpor a lógica matemática para o ambiente de desenvolvimento (IDLE), manipulando variáveis dinâmicas, controles de fluxo e estruturas de repetição. Mais do que digitar linhas de código, o foco é a preparação e o tratamento de dados utilizando bibliotecas nativas de Inteligência Artificial (Pandas, Numpy, Matplotlib), desenvolvendo a depuração técnica e a argumentação lógica de cada solução criada.

## Indicadores de Competência (O que será avaliado)

Para atingir a aprovação nesta Unidade Curricular, o estudante deve demonstrar o alcance dos seguintes indicadores profissionais previstos no perfil de conclusão:

## 🗺️ Estrutura do Cronograma (67 Horas)

A nossa UC está dividida estrategicamente em **cinco grandes Situações de Aprendizagem (SAs)**, aliando o rigor sintático do Python à rotina prática de negócios:

### [SA 1 - O "Hello World" dos Negócios]() (12h)
* **Foco:** Ambiente IDLE, Tipagem Dinâmica, Variáveis, Constantes e I/O (Input/Output).
* **Metodologia:** *Pair Programming* (Programação em Duplas).
* **Desafio Prático:** Criar um script inicial de cadastro com entrada, processamento matemático simples e saída de dados.
* **Ação Final:** Demonstração cruzada do script e explicação de conversão de tipos (casting).

### [SA 2 - Tomada de Decisão Lógica e Auditoria]() (15h)
* **Foco:** Expressões, Operadores e Estruturas Condicionais simples e compostas (if, elif, else).
* **Metodologia:** Rotação por Estações + Fluxogramas.
* **Desafio Prático:** Codificar uma árvore de decisão para aprovação de crédito ou simulação de investimentos.
* **Ação Final:** Defesa oral do código ("Post-it Único"), exercitando a leitura técnica do recuo e indentação do Python.

### [SA 3 - Automação Massiva e Coleções de Dados]() (15h)
* **Foco:** Estruturas de repetição (for, while, range, break), listas, tuplas e dicionários.
* **Metodologia:** Aprendizagem Baseada em Projetos (PBL) + *Scrumlite*.
* **Desafio Prático:** Construir um menu interativo e dinâmico que gerencie um estoque fictício utilizando dicionários.
* **Ação Final:** *Sprint Review* do mini-sistema rodando sem falhas de loop infinito.

### [SA 4 - Modularização e Confiabilidade de Software]() (10h)
* **Foco:** Criação de funções (def) e Otimização de Código com tratamento de exceções (try/except).
* **Metodologia:** Programação Guiada + Resolução de Problemas.
* **Desafio Prático:** Refatorar scripts antigos para funções isoladas e testar vulnerabilidades (*Bug Bounty Educacional*).
* **Ação Final:** Auditoria cruzada onde os alunos tentam gerar falhas (`Crash`) no sistema das equipes concorrentes.

### [SA 5 - Laboratório Base de Dados para IA]() (15h)
* **Foco:** Manipulação de *datasets* volumosos com Pandas, NumPy e Matplotlib.
* **Metodologia:** Laboratório de Dados + Estudo de Caso.
* **Desafio Prático:** Importar um arquivo CSV e realizar análise exploratória, limpeza de dados nulos e formatação.
* **Ação Final:** Criação de um *dashboard*/gráfico final e apresentação da descoberta ao instrutor.

## 💎 Marcas Formativas do SENAC em Evidência

Este projeto pedagógico foi desenhado para consolidar o seu perfil profissional por meio das seguintes marcas formativas institucionais:
* **Domínio técnico-científico:** Aplicação das convenções que padronizam a linguagem Python (PEP 8) e domínio de bibliotecas como Pandas e NumPy para manipulação estruturada de dados.
* **Visão crítica:** Prática constante de *Code Review* falado, depurando lógicas condicionais e avaliando vulnerabilidades nos sistemas.
* **Colaboração e Comunicação:** Execução da metodologia de *Pair Programming* e uso exclusivo do "Post-it Único" para defender as escolhas algorítmicas de forma autônoma e persuasiva.
* **Autonomia digital:** Configuração do ambiente IDLE, manipulação autônoma de diretórios e extração de *insights* via *Data Wrangling*.
* **Protagonismo econômico e Atitude Sustentável:** Foco no desenvolvimento de códigos robustos e confiáveis que geram valor para cenários reais de negócio.
